System Design
REEM Architecture
📡

fNIRS Sensor Layer

Non-invasive near-infrared spectroscopy measures hemodynamic activity in the prefrontal cortex — the region first affected by Alzheimer's.

🤖

CNN-LSTM Model

Convolutional layers extract spatial EEG features while LSTM networks capture temporal patterns, achieving 98.67% detection accuracy.

🔐

Federated Learning

Patient data never leaves the device. Only encrypted model updates are shared — ensuring full GDPR compliance and privacy.

⚡

tES + Bioenergy

Transcranial electrical stimulation supports cognitive function while bioenergy harvesting powers the device sustainably.
