The Research
About REEM
A non-invasive, privacy-preserving AI diagnostic system for early Alzheimer's detection — built at the intersection of neuroscience, deep learning, and federated computing.
Research Question
The Core Question
"Can a non-invasive wearable system integrating fNIRS-based hemodynamic signals with a hybrid CNN-LSTM model and Federated Learning achieve clinically significant accuracy in early-stage Alzheimer's detection while preserving patient data privacy?"
Hypothesis
A wearable fNIRS system combined with CNN-LSTM trained using Federated Learning will achieve ≥85% classification accuracy in distinguishing Healthy Controls, MCI patients, and early Alzheimer's patients — while maintaining full data privacy through distributed training.
✓ Target: ≥85% accuracy  ·  Achieved: 98.67%

Component Details
Technical Deep Dive
📡
fNIRS Sensor Layer
Functional Near-Infrared Spectroscopy
Non-invasive optical imaging technique that measures hemodynamic responses in the prefrontal cortex — the first brain region affected by Alzheimer's pathology. Unlike MRI or PET, fNIRS is portable, silent, and wearable.
MethodNear-infrared light absorption
Target RegionPrefrontal Cortex
InvasivenessNon-invasive
Channels19 EEG channels
🤖
CNN-LSTM Model
Hybrid Deep Learning Architecture
Convolutional Neural Networks extract spatial feature maps from multi-channel EEG signals, while Long Short-Term Memory networks capture temporal dependencies across 128-timestep windows — achieving superior classification over either architecture alone.
Parameters214,531
Input Shape128 × 19
Output Classes3 (Healthy / MCI / AD)
OptimizerAdam · lr=0.001
🔐
Federated Learning
Privacy-Preserving Distributed AI
Instead of centralizing sensitive patient data, each device trains a local model and shares only encrypted gradient updates. The global model improves without ever exposing raw patient data — fully GDPR-compliant by design.
FrameworkFlower (flwr)
Data SharedEncrypted gradients only
ComplianceGDPR · HIPAA aligned
PrivacyLocal training only
⚡
tES + Bioenergy
Therapeutic & Sustainable Layer
Transcranial Electrical Stimulation delivers safe, low-intensity electrical currents to support cognitive function in affected brain regions — transforming REEM from diagnostic-only to a therapeutic intervention platform powered by bioenergy harvesting.
StimulationTranscranial Electrical (tES)
SafetyLow-intensity, non-invasive
Power SourceBioenergy harvesting
GoalCognitive support
